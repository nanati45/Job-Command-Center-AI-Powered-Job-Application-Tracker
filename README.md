## üöÄ How It Works

This is not just a job board; it is an intelligent agent.

1.  **Ingestion:** An **n8n** automation workflow runs hourly, fetching jobs from various RSS feeds and API endpoints.
2.  **Analysis (AI Agent):**
    - **Extraction:** GPT-4o-mini cleans the raw HTML/XML to extract structured data (Salary, Tech Stack, Benefits).
    - **Scoring:** The AI compares the job description against my **parsed resume text** and assigns a Match Score (0-100).
    - **Reasoning:** It generates a 1-sentence explanation of _why_ I am (or am not) a good fit.
3.  **Generation:** If the score is >70, the AI writes a bespoke cover letter tailored to the specific company and role.
4.  **Storage:** Data is pushed to **Supabase** (PostgreSQL).
5.  **Frontend:** A modern **React (Vite)** dashboard listens to Supabase Realtime events to update the UI instantly without refreshing.

````sql

  id bigint generated by default as identity primary key,
  created_at timestamp with time zone not null default now(),
  title text null,
  company text null,
  description text null,
  link text null,
  score numeric null,
  reasoning text null,
  benefits jsonb null,
  cover_letter text null,
  status text not null default 'new'::text
);
